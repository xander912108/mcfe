# Mentori Club — Контекст для продолжения

## Что уже реализовано

### Страницы (работают, готовы к продакшену)

#### 1. Страница "Главное сейчас" (`/leader`)
- Путь: `src/LeaderConsoleMain.tsx`
- Секции: заголовок, human summary, Пульс сообщества (74/100), карточки "Ваше внимание сегодня" (3 шт.), "Сообщество справляется само", "Что можно передать сообществу", "Риски"
- Модальное окно "Пульс сообщества" с подробной разбивкой по параметрам
- Правый сайдбар с "Советником" (можно скрыть/показать)
- Toast-уведомления подключены, кнопки навигации рабочие
- Анимации section-fade-in на всех секциях

#### 2. Страница "Вступление" (`/leader/entry`)
- Путь: `src/LeaderConsoleEntry.tsx` (~2300 строк)
- **Заявки** (4 фильтра): Требуют ответа, На уточнении, Одобрены, История
  - 9+ заявок с реалистичными данными
  - Side-panel для просмотра деталей заявки (slide-in-right, full height, z-50)
  - Модалки действий: Одобрить, Задать уточнение, Не принимать сейчас, Вернуть в работу
  - Confirm-модалки при выходе без сохранения
- **Новички** (3 фильтра): Все новички, Ждут первой связи, Стартовый путь
  - 5 новичков с разными статусами
  - Side-panel для просмотра новичка с блоком "Первый вопрос новичка"
  - Стартовый путь новичка (4 шага) с прогрессом
- **Архив** — отдельный фильтр с архивными заявками
- **Поиск** по заявкам и новичкам
- Единый стиль карточек: плоские строки, золотой hover-border
- Side panel = контекст (просмотр), Modal = действие

### Toast-уведомления
- Путь: `src/ToastContext.tsx`
- 4 типа: success, error, warning, info
- Хук `useToast()` доступен в любом компоненте
- Используется в LeaderConsoleMain для всех кнопок действий

### Общая инфраструктура
- `src/App.tsx` — роутинг, хедер, сайдбар, переключение dark/light mode
- `src/index.css` — CSS-переменные, анимации (slide-in-right, modal-enter, section-fade-in, toast), hover-утилиты
- Тёмная/светлая тема через CSS-класс `.dark`
- Мобильная адаптация (нижняя навигация)
- Комьюнити-селектор в хедере с дропдауном

## Архитектурные паттерны (ВАЖНО соблюдать)

### Side panel vs Modal
- **Side panel** (`z-50`, `slide-in-right`) = контекст, просмотр, информация
- **Modal** (`z-60`, `modal-enter`, backdrop) = действие, форма, решение
- Никогда не дублировать side panel модалкой

### Кнопки в модалках
- Паттерн: `[Действие]` (золотой/терракотовый) / `[Вернуться к заявке]` (серый)
- НЕ использовать кнопку "Отмена" — она сбивает с толку
- Confirm-модалка (`z-70`) при выходе с несохранёнными изменениями

### Стили карточек
```tsx
// Единый паттерн
<div className="hover-border-gold transition-all duration-200 ..." style={{ border: '1px solid transparent' }}>
```

### Цвета
- Gold: `#C9A96E` (основной акцент)
- Terracotta: `#C9706A` (тревога, риск)
- Sage: `#6B9E7C` (успех, справляется)
